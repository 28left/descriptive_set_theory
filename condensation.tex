    \hypertarget{the-condensation-lemma}{%
\subsection{The condensation lemma}\label{the-condensation-lemma}}

```\{admonition\} Condensation lemma

There is a finite set \(T\) of axioms of \(\ZF - \text{Power Set}\) so
that if \(M\) is a transitive set with \(M\models T + \VL\), then
\(M = L_\lambda\) for some limit ordinal \(\lambda\). ```

    Define: \$\varphi\_\{\VL\} = \$ conjunction of the axioms in \(T\) and
the axiom \(\VL\).

    \hypertarget{first-application-vl-implies-gch}{%
\subsection{\texorpdfstring{First application: \(\VL\) implies
\(\GCH\)}{First application: \textbackslash VL implies \textbackslash GCH}}\label{first-application-vl-implies-gch}}

    ```\{admonition\} Key Lemma

Suppose \(V=L\). If \(\kappa\) is a cardinal and \(x \subseteq \kappa\),
then \(x \in L_{\kappa^+}\). ```

    

    Guide to proof: \emph{Verify each of following steps.}

    \begin{itemize}
\item
  There exists limit \(\lambda > \kappa\) such that \(x \in L_\lambda\)
  and such that \(L_\lambda \models \varphi_{\VL}\)
\item
  Let \(X = \kappa \cup \{x\}\).\\
  \emph{What do we know about \(X\) at this stage?}
\item
  There exists an \textbf{elementary substructure}
  \(N \preceq L_\lambda\) such that \begin{equation*} \tag{$*$}
    X \subseteq N \subseteq L_\lambda \quad \text{ and } \quad |N| = |X|.
  \end{equation*} \emph{Use a famous theorem from logic.}
\item
  Can we apply the condensation lemma to \(N\)?\\
  \emph{What are the possible obstacles?}
\item
  Apply a Mostowski collapse to \(N\). This gives us a transitive \(M\)
  isomorphic to \(N\).\\
  \emph{Now we have a new problem. What is it? (Hint: where does \(x\)
  go?)}
\item
  Argue that the Mostowski isomorphism fixes \(x\).\\
  \emph{Hint: What does it do with ordinals?}
\item
  Now we can apply condensation. This yields \(M = L_\beta\) for some
  \(\beta\).\\
  \emph{Where does this put \(x\) now?}
\item
  \textbf{Key argument}: \(|\beta| = \kappa\).\\
  \emph{Hint: Proposition 53}
\item
  Finish: \(x \in L_{\kappa^+}\)
\end{itemize}

    ```\{admonition\} \(\VL \to \GCH\)

If \(\VL\), then for all cardinals \(\kappa\), \(2^\kappa = \kappa^+\).
```

    ```\{admonition\} Corollary

If \(\ZF\) is consistent, so is \(\ZF + \AC + \mathsf{GCH}\). ```

    \hypertarget{second-application-the-complexity-of-constructible-reals}{%
\subsection{Second application: the complexity of constructible
reals}\label{second-application-the-complexity-of-constructible-reals}}

    The set of all constructible reals is defined by a \(\Sigma_1\) formula
over set theory:

\[
   \varphi(x_0) \; \equiv \; \exists y \; [y \text{ is an ordinal }  \; \wedge \; x_0 \in L_y \; \wedge \; x_0 \text{ is a set of natural numbers }  ].
\]

    \textbf{Can we ``convert'' this into a formula of second order
arithmetic?}

    \emph{How could the condensation lemma help with this?}

    \textbf{Key ideas}:

\begin{itemize}
\item
  every constructible real shows up at a countable stage of \(L\).\\
  \emph{Why?}
\item
  Hence if \(\alpha \in L \cap \Baire\), there exists a countable
  \(\xi\) such that \(x \in L_\xi\).
\item
  Then \(L_\xi\) is countable, too.\\
  \emph{Why?}
\item
  Hence we can hope to replace \(L_\xi\) by something like
  \textgreater{} ``\emph{there exists a real that codes a model that
  looks like \(L_\xi\)}''
\end{itemize}

    \textbf{Key ingredients}:

\begin{itemize}
\tightlist
\item
  Condensation lemma and Mostowski collapse\\
  \emph{Can you think why these are important here?}
\end{itemize}

    \[
\begin{multline*} \tag{$**$}
    \alpha \in L \cap \Baire \iff \exists \beta \exists m \: [E_\beta \text{ is  extensional and well-founded} \\ \: \wedge \: (\omega,E_\beta) \models \phi_{\VL} \: \wedge \: \pi_\beta(m) = \alpha ],
\end{multline*}
\] where \(\pi_\beta\) is the Isomorphism of the Mostowski collapse of
\(E_\beta\).
